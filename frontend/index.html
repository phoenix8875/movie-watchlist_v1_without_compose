3 Tier Docker App for Movie Watchlist
    ⚙️ LIVE COMPONENT INTERACTION FLOW ⚙️
    [ Your Browser ]
    Executes script.js locally in RAM
    Sends Data -> EC2 Public Port 5000
    │
    ▼ (Public Internet Traffic)
    [ Frontend Container: Nginx ]
    Container Name: watchlist-frontend
    Maps: Host Port 80 -> Container Port 80
    *Delivered the HTML/JS files to your client
    │
    ▼ (Data bypasses Nginx direct to API)
    [ Backend Container: Node.js API ]
    Container Name: watchlist-backend
    Maps: Host Port 5000 -> Container Port 5000
    *Validates request & constructs SQL queries
    │
    ▼ (Internal Network: watchlist-net)
    [ Database Container: PostgreSQL ]
    Container Name: postgres-db
    Internal Port: 5432 (Host Port: CLOSED)
    *Permanently saves data record to disk
